What it is
==========
Parser for Simple Data Interchange Format [SiDIF](http://wiki.bitplan.com/index.php/SiDIF)

Installation
============
```bash
pip install py-sidif
```

Get Sources
===========
```bash
git clone https://github.com/WolfgangFahl/py-sidif
cd py-sidif
scripts/install
```

Testing
=======
```bash
scripts/test
```
